# 2026 Customer Support Policy

Refunds can be requested within 7 days of payment. A refund is not available if the account has already used 50% or more of the monthly quota.

환불은 결제 후 7일 이내에 요청할 수 있습니다. 월 사용량이 50% 이상이면 환불이 제한됩니다.

The Free plan does not support file uploads. The Pro plan supports up to 1,000 file uploads per month.

무료 플랜에서는 파일 업로드가 지원되지 않습니다. Pro 플랜에서는 월 1,000건까지 파일 업로드가 가능합니다.

Customer names, email addresses, and phone numbers must be masked before storage. Raw personal data should not be shown in analytics dashboards.

고객 이름, 이메일 주소, 전화번호 같은 개인정보는 저장 전에 마스킹해야 합니다. 원본 개인정보는 분석 대시보드에 노출하면 안 됩니다.

If a user asks about a policy that is not written in this document, the support agent should say that the document does not provide enough information.

문서에 없는 정책을 물어보면 지원 에이전트는 문서에서 확인할 수 없다고 답해야 합니다.
