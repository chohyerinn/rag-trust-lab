# 서버 점검 공지

서버 점검은 일요일 오전 2시부터 5시까지 진행됩니다.

점검 시간에는 일부 접수 기능과 이메일 알림이 지연될 수 있습니다.

이 문서는 소비자 상담 운영시간이나 피해구제 절차의 근거가 아닙니다.
